# Sales Data ETL Pipeline

## Project Overview
This project demonstrates an end-to-end ETL (Extract, Transform, Load) pipeline for processing sales data.

The pipeline performs the following tasks:

1. Extract raw sales data from CSV files
2. Transform and clean the data using Python (Pandas)
3. Load the processed data into PostgreSQL
4. Schedule the pipeline using Apache Airflow
5. Store raw and processed data in AWS S3

## Architecture

Raw CSV Data
↓
Python ETL Scripts
↓
PostgreSQL Database
↓
Airflow Scheduler
↓
Analytics Dashboard

## Technologies Used

- Python
- Pandas
- SQL
- PostgreSQL
- Apache Airflow
- AWS S3
